Set Table Columns

SET — Number, or tag: W warmup, F failure, D drop
KG / REPS — Editable inputs. Pre-filled from previous workout.
EFF. — Effective resistance (only shown for exercises with a resistance profile). Auto-computed.
RPE — Rate of perceived exertion (optional, 6-10)
✓ — Tap to complete set & start rest timer. ★ = PR detected!

Effective Resistance

The purple badge shows when an exercise has a resistance profile configured.

User enters loaded weight (plates on the machine). The EFF. column auto-computes true resistance.

Charts use effective weight for apples-to-apples comparison.

Rest Timer

Appears between exercises after completing a set. Shows countdown, +30s extend, and skip.

Also fires a system notification for background use.

Supersets

Grouped exercises shown with an orange left border and label. Exercises within alternate during rest.

Bottom Navigation

5 tabs: Workout, History, Exercises, Templates, Body

Analytics and Settings accessible from top-bar or Exercises tab.
